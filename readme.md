## how to

+ zero geometry origin (using **blender**):
  + delete default cube
  + import geometry (`.stl`, `.obj`, etc.)
    + if not already selected, select (`shift+T`)
    + right click on STL -> Set Origin -> Origin to Geometry
    + Then in side window set origin to (0,0,0)
